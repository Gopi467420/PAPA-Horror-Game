How to Download and Run the Game

1. Download the ZIP File: Go to the GitHub repository and download the entire project as a ZIP file.
2. Download the Large File Separately: Download the sharedassets0.assets.resS file from this (https://github.com/Gopi467420/PAPA-Horror-Game/blob/main/PAPA_Data/sharedassets0.assets.resS).
3. Replace the File: After extracting the ZIP file, navigate to the PAPA_Data directory and replace the existing sharedassets0.assets.resS file with the one you just downloaded.
4. Start the Game: Go back to the main folder and open the " PAPA " file to start the game.
